Create User Project
📌 About The Project

Create User is a simple and interactive web application where users can create their own profile by entering personal information.
After clicking the Create User button, the user can add their details, and a personal user card/page will appear on the screen.

The application also keeps track of how many users have been created with a live Number of Users counter.

✨ Features
👤 Create new users
📝 Add first name and last name
📍 Add address/location
🖼️ Upload profile image
🎂 Enter age
💼 Add job/profession
🚻 Select gender
📄 Display user profile card
🔢 Live user counter
📊 Real-time number of users tracking
⚙️ How It Works
Click the Create User button
Fill in the required information:
Profile Image
First Name
Last Name
Age
Address
Job
Gender
Submit the form
The application will:
Create a user profile card
Display the entered information
Increase the user count automatically
